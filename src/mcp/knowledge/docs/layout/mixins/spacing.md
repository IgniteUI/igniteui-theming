# spacing() mixin

Defines spacing custom properties used by pad(), pad-inline(), and pad-block().

## Sass example
```scss
@include spacing();
```

## CSS example
```css
:root {
  --ig-spacing: 1;
  --ig-spacing-inline: 0.75;
  --ig-spacing-block: 0.5;
}
```

## Notes
- spacing() is typically included once at the root scope.
