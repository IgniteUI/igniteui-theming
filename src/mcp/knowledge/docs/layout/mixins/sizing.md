# The sizing() mixin

Defines [CSS @property](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/At-rules/@property) rules for size levels:
- --ig-size-small
- --ig-size-medium
- --ig-size-large

## Sass example
```scss
@include sizing();
```

## Notes
- Typically included once at the root scope/stylesheet.
